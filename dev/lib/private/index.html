LAP_julia.lap.multi_mat_div — Function
multi_mat_div(A, b)

Return E, where E[i, :] is the solutution of the least squares problem $\min\|D_(i)x - c_(i)\|^2$ for each $D_(i)$ and $c_(i)$, where $D_(i)$ is A[:, :, i], $c_(i)$ is b[:, i] and $i$ is the size of the second dimension of b and third dimension of A. In other words, each row of E is the solution to one matrix from A and it's corresponding vector from b.

function window_sum!(filter_result, pixels, image_size, window_size)

Filter the array pixels of shape image_size with a filter that performs a sum of values of the area window_size with each pixel as the center. Store the result in filter_result. The filtering uses "symmetric" padding.
